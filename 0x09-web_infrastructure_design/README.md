# Web Infrastructure Design

This team project introduces learners to the web's infrastructure design.

## Key Concepts and Resources
* Network basics
* Server
* Web server
* Application server
* DNS & DNS record types
* Load Balancer
* Monitoring
* Database
* Single point of failure
* HTTP & HTTPS
* Firewall
